# ARENA.md — адаптер для Arena.ai Agent Mode

> **Языки:** [English](ARENA.en.md) · Русский (текущий) · **Главный контракт:** [`AGENT.md`](../AGENT.md).

Этот файл — тонкий адаптер для Arena.ai Agent Mode. Он не вводит собственных правил, а направляет Arena.ai к общим контрактам проекта, не дублируя их.

- **Главный контракт, поведение и методология** — правило языка, **Spec → Verifier → Environment**, Автопилот / Спросить сначала / Никогда, порядок провайдеров и single source of truth по грекам: см. [`AGENT.md`](../AGENT.md).
- **Команды разработки, тестирование данных и проверки перед PR** — см. гид разработчика на [английском](DEVELOPMENT.en.md) или [русском](DEVELOPMENT.ru.md).
- **Инструкция по agentic-setup** (как повторить эту настройку в других проектах): см. [английский](Setup.en.md) или [русский](Setup.ru.md).
